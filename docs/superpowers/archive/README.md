# Superpowers Archive

Completed implementation plans and specs from past `/writing-plans` /
`/subagent-driven-development` sessions. Kept for historical context and
decision rationale; not actively maintained.

For current development workflow, see `docs/workflow.md`. For active work,
see [GitHub Issues](https://github.com/orgs/Bellese/projects/33/views/3).

**Note:** Cross-references inside archived plans (e.g., paths under
`docs/superpowers/plans/` or `docs/superpowers/specs/`) reflect the
directory layout at the time the plan was executed. Files were moved
into this `archive/` directory during the 2026-04-27 docs cleanup;
the archived plans were not rewritten because they are historical
records of what was planned and shipped.
